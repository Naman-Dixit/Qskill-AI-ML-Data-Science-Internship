## Tasks

### Task 1 — Iris Flower Classification

**Objective:** Build and compare multiple classifiers on the classic Iris dataset.

**What's covered:**
- EDA with scatter plots and histograms
- Train/test split and feature scaling with `StandardScaler`
- Three classifiers trained and evaluated:
  - Logistic Regression
  - K-Nearest Neighbors (k=5)
  - Decision Tree (Gini criterion)
- Evaluation using Accuracy, Confusion Matrix, and Classification Report

**Dataset:** `Iris.csv` (150 samples, 4 features, 3 species)  
**Libraries:** `scikit-learn`, `pandas`, `numpy`, `matplotlib`, `seaborn`

---

### Task 2 — Spam Mail Detector

**Objective:** Classify SMS messages as spam or ham using NLP and Naive Bayes.

**What's covered:**
- Text preprocessing — lowercasing, punctuation removal, stopword filtering (`nltk`)
- TF-IDF vectorization
- Multinomial Naive Bayes classifier
- Evaluation using Accuracy, Precision, Recall, and F1-Score
- Custom prediction function — test any message live

**Dataset:** `SMSSpamCollection` (UCI SMS Spam Collection)  
**Libraries:** `scikit-learn`, `nltk`, `pandas`
